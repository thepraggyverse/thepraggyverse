## Start Here

- **[CLI Printing Press](https://github.com/thepraggyverse/cli-printing-press)** - generates agent-first CLIs for APIs, with SQLite sync, offline search, and compound insight commands.
- **[Printing Press Library](https://github.com/thepraggyverse/printing-press-library)** - a tested library of CLIs generated by the Printing Press.
- **[agentcookie](https://github.com/thepraggyverse/agentcookie)** - keeps an agent machine's browser sessions synced from the Mac you actually use, encrypted over Tailscale.
- **[CodexBar](https://github.com/thepraggyverse/CodexBar)** - a macOS menu bar app for checking Codex and Claude Code usage stats without logging in.
- **[whisprlocal](https://github.com/thepraggyverse/whisprlocal)** - fully on-device voice-to-text for iOS and keyboard workflows, powered by WhisperKit and MLX.

## Current Focus

- Making coding agents more useful on real machines: authenticated sessions, local context, repeatable workflows, and boringly good CLIs.
- Building native Apple-platform tools with Swift, SwiftUI, and privacy-preserving local models.
- Turning rough personal workflows into polished utilities that other people can run.

## Working Style

I like tools that are:

- **Local-first** - useful without handing everything to a remote service.
- **Agent-ready** - documented, scriptable, inspectable, and designed for automation.
- **Small enough to understand** - sharp tools over sprawling platforms.
